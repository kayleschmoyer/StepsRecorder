# Roadmap

## MVP: Local Windows Steps Recorder

Goal:

Create a local-first Windows desktop app that records click-based steps, captures screenshots, allows editing, and exports DOCX/PDF.

### MVP Features

- Start recording
- Stop recording
- Capture clicks
- Capture screenshot of clicked monitor
- Store session locally
- Review steps
- Edit step title/description
- Delete steps
- Reorder steps
- Basic screenshot redaction/cropping
- Export DOCX
- Export PDF
- Settings page
- Friendly error handling
- Privacy reminder before export

## Phase 2: Better Editing

Potential features:

- Arrows
- Text labels
- Highlight boxes
- Blur tool
- Undo/redo stack
- Before/after screenshot versions
- Batch redaction
- Better crop handles

## Phase 3: Smarter Step Descriptions

Potential features:

- Detect active window title
- Detect clicked control where possible
- OCR screenshot text
- AI-generated step titles
- AI-generated workflow summary
- Auto-clean duplicate steps
- Suggested step descriptions

## Phase 4: Export Enhancements

Potential features:

- Custom document templates
- Company logo
- Header/footer options
- Jira-friendly export
- Xray test case export
- Markdown export
- HTML export
- Screenshot size options

## Phase 5: Team/Enterprise Features

Potential features:

- Team workspace
- Cloud sync
- Review/approval flow
- Shared templates
- Role-based permissions
- Audit history
- Enterprise policy controls

## Phase 6: Integrations

Potential integrations:

- Jira
- Xray
- Azure DevOps
- GitHub Issues
- Confluence
- SharePoint
- OneDrive
- Google Drive

## Not Planned for MVP

- Keystroke recording
- Audio recording
- Video recording
- Browser extension
- Mac/Linux support
- Cloud accounts
- Live collaboration
